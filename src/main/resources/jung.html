Jung View

Allgemein

Über Rechtsklick auf der Jung-View können neue Input bzw. Output Neuronen und Hidden hinzugefügt werden. Es gibt auch die Möglichlichkeit alle erzeugten Neuronen miteinander zu verbinden, wobei hier zwie verschiedene Verbindungsarten möglich sind, "vermascht" und "feedforward". Die Synapsen können wieder mit einem Rechtsklick gelöscht werden. Die Neuronen werden in Jung farblich dargestellt, Rot für Bias, Blau für die Input und Output Neuronen und Hellblau für Hidden Neuronen.
