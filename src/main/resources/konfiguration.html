Konfiguration

Topologie

Hier kann man auf individueller Weise ein eigenes Netz erzeugen. Man kann die Anzhal der Input, Output, Hidden Layer und Hidden Neuronen bestimmen Diese werden direkt an der Jung angezeigt, erst wenn das "Netzwerk erzeugen"-Button geklickt wird, kann das Netz trainiert werden. Über die Jung-Modis können entweder Input, Output oder Hidden Neuronen auf dem Jung-View hinzugefügt werden, wenn der Editing-Modi aktiviert wird. Wenn der Editing Modi aktiviert ist, wird mit einem Klick auf der Jung-View ein Neuron erzeugt, dieser kann ein Input, Output oder Hidden sein, je nachdem, welche RadioButton "Input", "Output" oder "Hidden" aktiviert ist. Mit der "Transforming-Modi" kann die Jung-View mit gehaltenenem Maus hin und her verschoben werden. Mit der "Picking-Modi" können einzelnen Neuronen selektiert werden, wobei mit SHIFT+Maus mehrere Neuronen selektiert werden können. Bei allen drei Modis wird mit dem Mausrad die Jung View vergrößert oder verkleinert. Mit dem Rechten Maustaste auf einem Nueron kann es gelöscht werden, oder Details anzeigen lassen, welche in der Standard-Einstellungen View (unten) zu sehen sind.

Standard Einstellungen

Über die Standard Einstellung-View können die Aktivierungsfunktion und die Initialgewichte für das Netz ausgewählt werden. Die Intialgewichte können zufällig zwischen den beiden eingegebenen Zahlen ausgewählt werden oder kann auch direkt ein exakter Gewicht angegeben werden.

Ausgewähltes Symbol

Diese View wird aktiviert wenn ein Neuronen in Jung View selektiert wird, (wie ein Neuron selektiert wird, kann aus der Topologie-Beschreibung entnommen werden) .Hier können bestimmte Einstellungen über das ausgewählte Neuron vorgenommen werden, wie z.B. Synapsengewicht oder auch die Aktivierungsfunktion ändern.
